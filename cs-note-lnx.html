Linux 기초: 실습용 핵심 키워드 정리

1. 네트워크 및 진단 명령어

  • ss: 소켓 통신 상태를 확인하는 최신 도구 (netstat 대체).
  • arp: IP 주소에 매핑된 물리적 MAC 주소 정보를 확인합니다.
  • tcpdump: 네트워크 인터페이스의 패킷을 실시간 캡처합니다.

2. 파일 시스템 및 권한 관리

  • /etc/passwd: 사용자 계정 정보가 담긴 파일입니다.
  • chattr: 파일의 속성을 변경하여 불변(Immutable) 상태 등으로 만들어 보안을 강화합니다.
  • setfacl: 보다 세밀한 사용자별 접근 제어 목록(ACL)을 설정합니다.

3. 서브넷 마스크 계산법 (Tip)

IP 주소 뒤의 /26과 같은 CIDR 표기법에 익숙해져야 합니다.

  • /26은 상위 26비트가 네트워크 영역임을 의미합니다.
  • 서브넷 마스크로는 255.255.255.192에 해당하며, 총 64개의 IP를 가집니다.

리눅스 시스템 관리 시험에서는 각 설정 파일(etc/ 하위)의 위치와 주요 명령어 옵션의 의미를 정확히 묻는 문제가 자주 출제됩니다.

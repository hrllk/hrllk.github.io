Netty: MessageToByteEncoderлҘј нҷңмҡ©н•ң мқёмҪ”л”© кё°мҙҲ

1. MessageToByteEncoderлһҖ?

MessageToByteEncoderлҠ” мһҗл°” кё°л°ҳмқҳ кі м„ұлҠҘ л№„лҸҷкё° л„ӨнҠёмӣҢнҒ¬ н”„л Ҳмһ„мӣҢнҒ¬мқё Nettyм—җм„ң м ңкіөн•ҳлҠ” нҒҙлһҳмҠӨмһ…лӢҲлӢӨ. л№„мҰҲлӢҲмҠӨ лЎңм§Ғм—җм„ң мӮ¬мҡ©н•ҳлҠ” л©”мӢңм§Җ к°қмІҙлҘј л„ӨнҠёмӣҢнҒ¬лЎң м „мҶЎн•ҳкё° мң„н•ҙ мқҙм§„ лҚ°мқҙн„°(Byte)лЎң ліҖнҷҳн•ҳлҠ” м—ӯн• мқ„ мҲҳн–үн•©лӢҲлӢӨ.

2. мЈјмҡ” мҡ©лҸ„

  • Outbound лҚ°мқҙн„° мІҳлҰ¬: м„ңлІ„м—җм„ң нҒҙлқјмқҙм–ёнҠёлЎң лҚ°мқҙн„°лҘј ліҙлӮј л•Ң, мһҗл°” к°қмІҙлҘј л°”мқҙнҠё лІ„нҚј(ByteBuf)лЎң мқёмҪ”л”©н•©лӢҲлӢӨ.
  • л„ӨнҠёмӣҢнҒ¬ м „мҶЎ мӨҖл№„: ліҖнҷҳлҗң л°”мқҙнҠё лҚ°мқҙн„°лҠ” мөңмў…м ҒмңјлЎң л„ӨнҠёмӣҢнҒ¬ мҶҢмј“мқ„ нҶөн•ҙ мғҒлҢҖл°©м—җкІҢ м „лӢ¬лҗ©лӢҲлӢӨ.

3. мӮ¬мҡ© л°©лІ•

нҒҙлһҳмҠӨ мғҒмҶҚ л°Ҹ кө¬нҳ„

MessageToByteEncoder<T>лҘј мғҒмҶҚл°ӣм•„ encode() л©”мҶҢл“ңлҘј мһ¬м •мқҳн•©лӢҲлӢӨ.

public class MyEncoder extends MessageToByteEncoder<MyMessage> {
    @Override 
    protected void encode(ChannelHandlerContext ctx, MyMessage msg, ByteBuf out) {
        // 1. л©”мӢңм§Җ к°қмІҙлҘј л°”мқҙнҠё л°°м—ҙлЎң ліҖнҷҳ
        byte[] bytes = convertMessageToBytes(msg);
        
        // 2. м¶ңл Ҙмҡ© л°”мқҙнҠё лІ„нҚјм—җ кё°лЎқ
        out.writeBytes(bytes);
    }

    private byte[] convertMessageToBytes(MyMessage msg) {
        // к°қмІҙлҘј л°”мқҙнҠёлЎң м§Ғл ¬нҷ”н•ҳлҠ” лЎңм§Ғ кө¬нҳ„...
        return msg.toString().getBytes();
    }
}

лҸҷмһ‘ л°©мӢқ

Nettyмқҳ нҢҢмқҙн”„лқјмқём—җ мқҙ мқёмҪ”лҚ”лҘј л“ұлЎқн•ҙл‘җл©ҙ, л©”мӢңм§ҖлҘј м „мҶЎн•ҳлҠ” мқҙлІӨнҠёк°Җ л°ңмғқн•  л•Ң мһҗлҸҷмңјлЎң encode() л©”мҶҢл“ңк°Җ нҳём¶ңлҗҳм–ҙ к°қмІҙк°Җ л°”мқҙнҠёлЎң ліҖнҷҳлҗң нӣ„ м „мҶЎлҗ©лӢҲлӢӨ.

NettyлҠ” лҢҖк·ңлӘЁ нҠёлһҳн”Ҫмқ„ мІҳлҰ¬н•ҳлҠ” л¶„мӮ° м„ңлІ„лӮҳ кІҢмһ„ м„ңлІ„ л“ұм—җм„ң мІҳлҰ¬ мҶҚлҸ„мҷҖ мһҗмӣҗ нҡЁмңЁмқ„ к·№лҢҖнҷ”н•ҳкё° мң„н•ҙ л„җлҰ¬ мӮ¬мҡ©лҗ©лӢҲлӢӨ.

# PRD: CS 지식 축적을 위한 블로그 포스트 시리즈

## 목표
자바/스프링 개발자로서 백엔드 역량을 강화하기 위해 Computer Science 핵심 영역을 체계적으로 학습하고, 이를 블로그 포스트로 정리한다.

## 핵심 요구사항
- 상위 목표 Task 1개로 관리한다.
- 하위 Task는 약 30개로 구성한다.
- 주제는 특정 언어 종속성을 최소화하되, 실무 백엔드 관점에서 이해할 수 있어야 한다.
- 운영체제, 커널, 네트워크, 자료구조/알고리즘, 데이터베이스, 분산시스템, 보안, 컴파일러/런타임, 관측성 등을 포함한다.

## 성공 기준
- 30개의 포스트 주제가 중복 없이 구성된다.
- 각 포스트는 문제의식, 핵심 개념, 실무 적용 포인트를 포함한다.
- 시리즈를 완주하면 시스템 설계/성능/안정성 관점에서 의사결정 품질이 향상된다.
